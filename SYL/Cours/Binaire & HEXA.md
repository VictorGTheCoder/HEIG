**Hexa to Dec**
A = 10, B = 11, C = 12, D = 13, E = 14, F = 15
A8CE = 10\*16^3 + 8\*16^2 + 12\*16^1 + 14\*16^0
= 40960 + 2048 + 192 + 14
= 43214

**Binaire to Hexa**

	N16= 0x96C4
