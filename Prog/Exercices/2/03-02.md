int i = 5, j = 11;
double x = 5, y = 11;

double m = 0;
```

```
1. m = j / i = 2.0
```
```
2. m = y / x = 2.2
```
```
3. m = j / i + 1.0 = 3.0
```
```
4. m = y / x + 1 = 3.2
```
```
5. m = y / x + j / i = 2.2 + 2 = 4.2
```
```
6. m = i + y / x = 5 + 2.2 = 7.2
```
```
